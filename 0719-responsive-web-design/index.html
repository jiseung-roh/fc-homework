Responsive Web Design
Step by Step Guideline

Responsive Web Design — a.k.a. RWD — plays crucial role in modern websites in terms of N Screen compatibility. Below is a simple guide to develop a basic responsive website via fluid grid, flexible image & video and media query.

Step 1.
Choose the breakpoint.

First of all, you need to choose the width sizes of the sites that render different CSS styles. These points are called "breakpoints" — breakpoints are the point which the content of your sites responds to provide the user with the optimized layout to consume the information. Google has provided a great reference that suggests the width size of breakpoints depending on device width and and how many columns are desirable accordingly.

The breakpoints that I personally prefer are 600px and 960px. Width under 600px is for mobile view, value between 601px and 960px for tablet and width over 961px for desktop view. Sometimes I add another breakpoint at 1200px for wide screen users.

Step 2.
Build a fluid grid system.

After you finish deciding on which point the website will break, it is time to build a fluid grid system accordingly. What is important here is that it is much more efficient to start your sites design from mobile first view and other devices later. The screen size of the mobile is small so that the content of the sites has to be compact. It is more recommended to put the core content first and add other subsidiary content in respect of the viewport size. This way of approach is called "Progressive Enhancement".

Building a fluid grid system is extremely simple. All you need to do is to set the width value using percentage(%) instead of fixed value like pixel or em/rem.

Let's make a grid system for mobile view first. I will use 4 columns.

/* module to clear the float properties. */
.row:after, .row::after {
  content: '';
  display: block;
  clear: both;
}
/* Float the grid items */
[class*="col-1"] {
  float: left;
}
/* 4 Columns, 's' indicates it's for mobile view */
.col-s-1-4 { width: 25%; }
.col-s-2-4 { width: 50%; }
.col-s-3-4 { width: 75%; }
.col-s-4-4 { width: 100%; }

I will finish making the grid system. I am going to use 8 columns for tablet and 12 for desktop view.

/* 8 Columns, 'm' for tablet view */
.col-m-1-8 { width: 12.5%; }
.col-m-2-8 { width: 25%; }
.col-m-5-8 { width: 37.5%; }
.col-m-4-8 { width: 50%; }
.col-m-5-8 { width: 62.5%; }
.col-m-6-8 { width: 75%; }
.col-m-7-8 { width: 87.5%; }
.col-m-8-8 { width: 100%; }

/* 12 Columns, 'l' for tablet view */
.col-l-1-12 { width: 8.33333%; }
.col-l-2-12 { width: 16.66666%; }
.col-l-3-12 { width: 25%; }
.col-l-4-12 { width: 33.33333%; }
.col-l-5-12 { width: 41.66666%; }
.col-l-6-12 { width: 50%; }
.col-l-7-12 { width: 58.33333%; }
.col-l-8-12 { width: 66.66666%; }
.col-l-9-12 { width: 75%; }
.col-l-10-12 { width: 83.33333%; }
.col-l-11-12 { width: 91.66666%; }
.col-l-12-12 { width: 100%; }

Step 3.
Set the class value to HTML element.

Lastly, you have to set media query value to make your fluid grid work. In order to do so, you have to set the grid class to the HTML element. But first, you need to add <meta> tag in <head> to optimize viewport size. 

<head>
  <title>Title of your website</title>
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>

Now you are really prepared to make your site responsive to the viewport size. Let's say you want 1 column-only design for mobile, 1 column in header and 2 columns in main for tablet, and 3 even columns for desktop view.

The mobile display uses 1 column only, meaning the width value of the element is 100%. The HTML markup would be like this: 

<div class="row">
  <header class="box-1 col-s-4-4"><header>
  <section class="box-1 col-s-4-4"><section>
  <section class="box-1 col-s-4-4"><section>
</div>

The tablet display is a little bit more complex than the mobile version but there is nothing difficult. The header takes up 100% width value and others take 50% each. 

<div class="row">
  <header class="box-1 col-s-4-4 col-m-8-8"><header>
  <section class="box-1 col-s-4-4 col-m-4-8"><section>
  <section class="box-1 col-s-4-4 col-m-4-8"><section>
</div>

All blocks share same width value, 33.3333%. 

<div class="row">
  <header class="box-1 col-s-4-4 col-m-8-8 col-l-4-12"><header>
  <section class="box-1 col-s-4-4 col-m-4-8 col-l-4-12"><section>
  <section class="box-1 col-s-4-4 col-m-4-8 col-l-4-12"><section>
</div>
